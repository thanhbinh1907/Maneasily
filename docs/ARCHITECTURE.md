# Kiến trúc & hướng dẫn ngắn

- src/css: styles toàn cục
- src/js: entry JS, sau này chia thành modules/components
- src/components: component UI (mỗi component 1 thư mục)
- src/pages: trang tĩnh nếu cần (html partials)
- src/services: gọi API / business logic
- src/assets: ảnh / icon / font
- tests: unit tests
